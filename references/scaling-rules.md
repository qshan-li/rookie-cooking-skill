# 份量缩放规则

缩放不是所有参数等比例放大。主料可以线性缩放，调味和时间需要分项处理。

| 项目 | 缩放方式 | 说明 |
| --- | --- | --- |
| 主料 | 线性 | 2 人份到 4 人份通常乘 2。 |
| 盐 | 先乘 0.75-0.9 | 大份菜更容易累积咸味，先少放，出锅前校正。 |
| 生抽 | 先乘 0.75-0.9 | 同时增加咸味和水分，避免等比例导致过咸。 |
| 糖 | 先乘 0.8-1.0 | 酸味菜可接近线性，红烧类按收汁程度调整。 |
| 油 | 不完全线性 | 以能覆盖锅底和润滑食材为准。 |
| 水/汤 | 视蒸发面积和炖煮时间 | 炖煮类不简单乘倍数，先保证液面高度。 |
| 炒制时间 | 不线性 | 大份量优先分批，单批时间基本不变。 |
| 炖煮时间 | 主要由食材厚度决定 | 数量增加不代表成倍延长，块更大才延长。 |

## 分批规则

- 快炒肉丝、青菜、蛋类：锅中食材超过锅底 70% 面积时，优先分批。
- 分批时每批调味少量预留，最后合并校正。
- 分批比延长炒制时间更能避免出水和口感变老。

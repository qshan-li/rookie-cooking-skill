# rookie-cooking-skill

厨房新手友好的做菜 Skill 设计稿。

目标是把中餐菜谱里常见的“适量”“大火”“炒熟”“收汁”等模糊经验，整理成能执行、能打印、能复盘的厨房文档：克数、时间、火力描述、状态判断、失败诊断、替代方案和原理解释都要清楚。

## Name

`rookie-cooking-skill` 保留了“厨房小白 / rookie”的谦逊定位：它不是大师菜谱库，而是一套帮新手稳定下厨的执行系统。

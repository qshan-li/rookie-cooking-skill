# Project: rookie-cooking-skill

## Overview

This project contains the design and requirements for a cooking Skill aimed at kitchen rookies. The Skill turns vague recipe instructions into executable kitchen documents with measurements, timing, state checks, failure diagnosis, substitutions, and cooking principles.

## Tech Stack

Documentation-first project. No runtime stack is defined yet.

## Architecture

Requirements live in `docs/`. Future implementation should preserve the documentation-first structure and keep recipe templates, principles, references, and generated Skill instructions separate.

# 食品安全温度

- ID：`food-safety-temperature`
- 名称：食品安全温度
- 一句话解释：温度计是判断肉、蛋、海鲜和剩菜安全性的最可靠工具；无温度计感官判断只能作为保守替代。
- 适用菜谱：
  - `qingjiao-rousi` → `recipes/meat/qing-jiao-rou-si.md` — 验证点：肉丝变白不等于熟透，用温度计确认中心达 71°C
  - `steamed-egg` → `recipes/soup/zheng-dan-geng.md` — 验证点：蛋羹中心温度达 74°C 时凝固安全，表面不晃动
  - `hongshaorou` → `recipes/meat/hong-shao-rou.md` — 验证点：炖肉中心温度达 85°C 以上时胶原充分溶出
- 来源：USDA FSIS Safe Minimum Internal Temperature Chart；FDA Food Code 2022；中国 GB 31654-2021《食品安全国家标准 餐饮服务通用卫生规范》；FoodSafety.gov Safe Minimum Internal Temperatures。

## 原理描述

致病微生物是否被降低到安全水平，取决于食材中心达到的温度和持续时间。家庭菜谱为了简单和安全，应优先使用官方最低内部温度；没有温度计时，必须说明感官判断可靠性较低，并采用更保守的加热策略。

## 关键变量

| 变量 | 会影响什么 | 新手判断 |
| --- | --- | --- |
| 中心温度 | 安全性 | 温度计插入最厚处中心，避免只测表面。 |
| 厚度 | 熟化时间 | 厚块比薄片需要更久，不能只看外表。 |
| 复热均匀度 | 剩菜风险 | 汤汁沸腾不代表厚块中心已热透。 |
| 人群风险 | 安全裕度 | 儿童、孕妇、老人、免疫低下者更保守。 |

## 实际应用

- 禽肉和含禽肉剩菜按 74°C / 165°F 目标处理。
- 猪牛羊整块肉通常以 63°C / 145°F 并静置为底线；绞肉和肉馅风险更高，按 71°C / 160°F。
- 鱼类按 63°C / 145°F，或肉质不透明且能用叉子分开；无温度计时仍要提示可靠性限制。
- 蛋菜按完全凝固或官方蛋菜温度目标处理，高风险人群避免流心蛋。

## 常见误区

- 误区：肉变白或蛋凝固就绝对安全。  
  修正：颜色和凝固只是线索，不能替代中心温度。

- 误区：多炒几下就能修复所有存放问题。  
  修正：变质、异味、发黏或存放不明的高风险食材应丢弃。
